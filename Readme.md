*********************   HTTP   ****************

it is the language that help in tranfer the data over the internet 

HTTP works on a very simple cycle called the Request-Response model.

HTTP header:-

these are the metadata that come with the http request and respnose.
it comes as the keyvalue pair like: name: umar
it is used for the authentication,cashing ,state managment

-request header: from client
-response header: from the server
-representaion heder:encoding/compresssion
-payload header:data



HTTP METHODE:-


GET: retrieve a request
HEAD: no message body(response header only)
OPTION:what is operation are available
TRACE:loopback test
DELETE:remove a resource
PUT:replace a resource
POST:add a resource
PATCH:change the part of resource



HTTP STATUS CODE:-

1XX:- INFORMATIONAL
2XX:-SUCCESS
3XX:-REDIRECTION
4XX:-CLIENT ERROR
5XX:-SERVER ERROR
